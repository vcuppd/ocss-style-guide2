Typography

The site is using Bootstrap v3.3.7. Any custom styles added to the site will defined below.

Links and Return to Top Links

The links have a custom color and font-weight.

a { color: #1B5E20; font-weight: 700; }

Horizontal Rule and Frequently Asked Questions Horizontal Rule

There are custom styles applied to the standard horizontal rule and the horizontal rule for the Frequently Asked Questions section.

.faq { border: 1px #fff solid; }

hr { border: 1px #000 solid; }

Buttons

The classes .btn, .btn-success, .btn-lg, and .btn-block, and role="button" are applied to two links styled as buttons on the website. Due to the role being set to button, the links will be recognized for accessibility. The first is in the Refresher section on the home page that leads to the OCSS test site. The second is located on the module pages that also leads to the OCSS test site.

Refresher Quiz
<a class="btn btn-success btn-lg btn-block" role="button">Refresher Quiz</a></code>

Table

The table has the Bootstrap styles .table, .table-striped, .table-condensed applied to the <table> element. The stripes in the table are customized to have background-color: #dff0d8; to go with the site color theme. There is also a Font-Awesome character [.fa .fa-check-square-o] used in the table to denote which modules should be completed for each role.
