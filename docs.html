Typography

Headings

The headings used for this site are the base Bootstrap headings h1 to h3

h1. Bootstrap heading Semibold 36px

h2. Bootstrap heading Semibold 30px

h3. Bootstrap heading Semibold 24px

Body Copy

Since the site is using the default Bootstrap styles, the default font-size for the is set to 14px, with a line-height of 1.428, and is applied to the body and all paragraphs. The default bottom-margin is set 10px.

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Consequuntur perspiciatis, laudantium. Nostrum libero quibusdam voluptates architecto, blanditiis quas voluptate adipisci exercitationem veniam unde iste itaque in? Aliquam assumenda, pariatur delectus.

Lead

The .lead class is added to the introduction paragraphs for the home page and module pages.

<p class="lead"> Lorem ipsum dolor sit amet .... </p>

Strong

The <strong> tag is used for emphasis in paragraphs.

<strong> Lorem ipsum dolor sit amet ... <strong>

Text Alignment

The following classes are used to center and right align copy

<p class="text-left"> Left aligned text. </p>
<p class="text-center"> Center aligned text. </p>
