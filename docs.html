Buttons

The classes .btn, .btn-success, .btn-lg, and .btn-block, and role="button" are applied to two links styled as buttons on the website. Due to the role being set to button, the links will be recognized for accessibility. The first is in the Refresher section on the home page that leads to the OCSS test site. The second is located on the module pages that also leads to the OCSS test site.

Table

The table has the Bootstrap styles .table, .table-striped, .table-condensed applied to the <table> element. The stripes in the table are customized to have background-color: #dff0d8; to go with the site color theme. There is also a Font-Awesome character [.fa .fa-check-square-o] used in the table to denote which modules should be completed for each role. There is also a drop shadow added to the table to give it a simple 3D look.

Nav Tabs

These use .nav, .nav-tabs, .tabe-content, .tap-pane, and .content classes to style the Frequently Asked Questions (FAQ) section on the home page. Each .tab-pane has a unique <id> so the data-toggle="tab" can be directed to the correct content.

Q. How do I view the slides in “full screen” mode and in “outline/notes” mode?

A. Click on the double arrow icon to go between “full screen” mode and “outline/notes” mode. When you click “notes,” you can view the script as it is narrated. The “outline” mode displays the title or main topic of the slides (slides with “screen” shots do not show words in the outline mode) and lets you navigate forward and backwards through slides.
Q. How do I stop the webinar if I need to take a break?

A. To pause the Webinar, click on the double parallel bars. When you are ready to proceed, click on the forward (arrow) button.

Panels

The panels use the Bootstrap class .panel .panel-success classes and are show on the home page as well as the module pages. A simple drop shadow is added to give the panels a 3D effect.
