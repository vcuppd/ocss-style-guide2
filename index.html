Options Counseling Style Guide

This is the style guide for the Virginia Options Counseling Statewide Standards Training website. It includes the basic global styles for typography, site colors, components like tables, panels, buttons, and tab panes, and the HTML and CSS guidelines.

Library Used

Bootstrap v3.3.7 was used to create this site along with several custom CSS styles applied to different areas of the site.
